设计迭代主线

十代反应器不是十次独立实验,而是一条持续收敛的工程改进链条 —— 每一代都在回应上一代暴露的问题。R6 是唯一一次被视频明确记录"失效"的世代,R7-R9 是针对这次失效的连续收敛过程,R10 目前证据不足以判断是否接入同一条主线。

R1 / 基线
固态阳离子交换树脂电解质
中间腔室(10mm,窗口1×1cm)填充强酸性阳离子交换树脂珠(粒径600–800μm)。阴极 CuO / H23C3 碳纸,阳极 Pt-Ru-C / H23C3 碳纸(CO 耐受型 HOR 催化剂,呼应阳极 H₂+1%CO 进气)。膜为 Sustainion X37-60(阴极侧)+ Nafion 117(阳极侧)。0.3 N·m 扭矩组装。
R2 → R3 / 渐进优化
给固态树脂通去离子水
结构不变,R2 在组装前对树脂做一次"预浸润"(pre-wetting),R3 进一步改成运行期间持续通入 DI 水 —— 目的都是提升固态电解质的离子电导率、降低初始电阻。两代验证了同一个假设的强化版本。
R4 / 路线切换
固态树脂 → 循环液态 1M KOH
narrator 明确点出"关键变化":中间腔室的固态电解质层被替换为持续循环的 1M KOH 液态电解质。同时膜型号从 Sustainion X37-60 降级到 X37-50,垫片从 0.2mm 增厚到 0.3mm。测试目的写得很直接:验证循环液态电解质能否提升电化学性能。
R5 / 减薄 + 加参比电极
腔室 10mm → 3mm,引入 Hg/HgO 参比电极
继续用液态 KOH,但把腔室厚度从 10mm 减到 3mm 以缩短离子传输距离、降低欧姆电阻、拉低电池电压;同时把氧化汞参比电极插入中间层,通过接线位置区分"阴极→中间腔室"电阻(R1)与"全电池"电阻(R3)。这一代还留下一套完整、可复用的拆解失效检查清单(见下文),但本身没有报告失效。
R6 / 扩窗尝试失效
腔室窗口 1×1cm → 3×3cm + 内部流道
保留 R5 的 3mm 腔室厚度,把窗口扩大到 3×3cm 并加一条内部流道,目标是增大电解质体积、改善分布,同时不牺牲离子传输距离。运行仅 10 分钟后即出现阴极水淹:narrator 当场判定"主要失效模式是阴极水淹,很可能是扩大后的腔室内液体积聚或压力不平衡导致"。拆解证实:阳极侧完全水淹、两侧 Sustainion 膜均有机械损伤(撕裂)、阴极表面出现蓝绿色沉积物(CuO 被电解质腐蚀的产物)。
R7 / 结构性简化解决水淹
彻底移除中间腔室,只保留一张膜
不再修补腔室几何形状,而是直接把中间腔室整个拿掉:阴阳极之间只隔一张 Sustainion X37-50 膜,用 1M KOH 浸润,不再有任何液态/固态电解质夹层。narrator 明确说目的是"简化结构、移除液态中间层、避免液体导致的水淹或压力失衡"。测试视频证实没有再出现阴极水淹,气体通路保持开放干燥 —— 这是本次记录里唯一一次被验证成功解决上一代失效模式的设计改动。代价:拆解发现膜偏干、催化剂有脱落粘连迹象,为 R8 埋下伏笔。
R8 / 隔膜换材料
AEM 离子交换膜 → 惰性 PTFE 隔膜(预浸润)
R7 测试后 AEM 膜破损并粘在催化层上,机械强度不足。R8 换用不导离子的纯物理 PTFE 隔膜,组装前用 1M KOH 预浸润一次以改善离子连接、降低初始电阻。拆解结果:不再水淹,但整体"干涸" —— 阳极、隔膜、阴极侧全部偏干,narrator 直接判定"一次性预加的电解质液滴不足以维持测试全程的水合"。失效模式从"水淹"变成了"缺水"。
R9 / 连续供水
引入"微液滴系统"持续给隔膜供水
用随气流连续输送的水微液滴替代 R8 那种一次性预浸润。拆解证实液滴成功送达阳极碳纸背侧(大量可见水滴),但穿不透多孔碳纸到达催化剂-隔膜界面,那里仍然干燥。narrator 把限制因素从"供水总量不够"重新定位为"碳纸传质路径效率不够",并明确提出下一步方向:改善气体通道到催化剂-隔膜界面的液体传输效率。
R10 / 平行的小型化装配路线仅1.Cell
夹具式小电池,教学录音,尚无测试/拆解素材
与 R1-R9 的流场板反应器结构不同,R10 用一套自制夹具("支撑 SELL 的架子")装配更紧凑的小电池,narrator 全程用中文讲解,且明确说明这是录给拍摄团队看的教学音轨("你们拍的时候不用说话,直接装就行"),不是随实验同步的组装记录。约 10 分钟即可装完一个电池,电极刻意裁小(0.5×0.5cm)以节省催化剂用量。这段素材首次触发了对中文转录的术语纠错("羊脊"→阳极、"音极"→阴极,均为同音异字误听)。是否是 R9 之后的正式迭代、还是并行的小型化验证平台,需要 2.Test/3.Post-test 素材补上后才能判断。

关键结论

这批反应器针对的是 CO 还原 / 氢氧化(CORR–HOR)串联电解池:阴极通 CO 气体做还原反应,阳极通 H₂+1%CO 做抗中毒氢氧化,中间层负责离子传导。R1–R6 的演进主线在回答"中间腔室该用什么方式导离子,才能同时兼顾电导率、离子传输距离和机械可靠性":R1–R3 强化"固态树脂+加水";R4 换成循环液态电解质;R5 把腔室做薄;R6 试图扩大容量但引发阴极水淹、腐蚀催化剂、撕裂膜,10 分钟内失效。

R7–R9 是针对 R6 失效的一条连续、自洽的收敛链条:R7 不再修补腔室几何形状,而是直接移除中间腔室,测试证实水淹问题确实消失,但拆解发现膜偏干、催化剂脱落粘连;R8 把 AEM 膜换成惰性 PTFE 隔膜(解决膜机械强度问题),但整体"干涸"成为新的失效模式;R9 引入连续"微液滴"供水,证实水能送达阳极气室,却穿不透多孔碳纸抵达真正需要水合的催化剂-隔膜界面 —— 问题从"供水总量不够"精确收敛到"碳纸传质路径效率不够"。三代迭代形成了一条教科书式的"发现失效模式 → 针对性改动 → 验证 → 暴露下一层瓶颈"的工程收敛过程,是本次批处理里信息密度最高的部分。R10 是一条尚未接入这条主线的平行小型化装配路线,目前只有 1.Cell 素材且是教学录音,暂无法判断与 R7-R9 主线的关系。

R7 的"移除中间腔室"是本次 35 段视频里唯一一次被完整验证成功解决上一代失效模式的改动 —— 测试阶段实时观察 + 拆解阶段物理证据双重印证,不再有阴极水淹。
R9 收尾时 narrator 明确提出的下一步方向是"改善气体通道到催化剂-隔膜界面的液体传输效率",这是当前证据链指向的直接改进方向,如果存在 R11,建议优先核对是否针对这一具体瓶颈(而不是笼统的"供水不足")做了改动。R10 补上 2.Test/3.Post-test 素材后,应优先确认它是否是这条主线的延续。

数据质量说明

35 段视频里有 8 段(#8、#9、#12、#13、#21、#23、#31、#35)最终产出 0 个知识节点,原因分三类:一类是纯标签特写或无人声的环境音段,Whisper 在这类音频上容易输出与画面无关的幻觉文本;一类是拆解片段里 narrator 用词过于碎片化,GPT-4o 判断不足以构成可靠的实体关系;R7-R10 新增了第三类——#31(R9 "1.Cell-same with R8")整段是无关的相机测试录音,#35(R10 第二段素材)完全没有音轨,后者触发了 pipeline 的一处修复:`stage1_asr` 现在会先检测音轨是否存在,没有音轨就直接记录空转录并继续跑完剩余阶段,不再让整个 batch 因为一段静音素材而中断。以上都是合理的保守输出或素材本身的性质,不是流程故障。

#20 与 #25(均为 R6,ASR 质量明显下降的两段)里出现了若干高度扭曲、难以可靠还原的催化剂/基材描述,判断为一次性误听、不具规律性,未强行加入词典以免过拟合。气体流量单位在不同视频里出现 sccm / ccm / fcl / SACM 多种写法混用,其中 SACM 经画面仪表验证确系 sccm 的误听已加入词典,ccm、fcl 因存在误伤已正确文本的风险仍保留原文。#33(R9 拆解)还发现了一处术语纠错的假阳性:已有的碳纸型号模糊匹配规则(阈值 0.82)把这次本来识别正确的通用表述"the carbon paper"也错误地插入了"H23C3"型号名 —— 内容本身仍然事实正确(这批实验确实全程用 H23C3 碳纸),但不完全对应"原文逐字讲了什么",已记录该已知局限,未改动匹配算法本身。
